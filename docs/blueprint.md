# **App Name**: LotoHub

## Core Features:

- Dark Header: Displays a dark background header with a home icon on the left, the 'LotoHub' title in the center, and a menu button on the right.
- Light Header: Shows a light background header with 'Saldo' (Balance) and its value on the left, 'Bônus' (Bonus) and its value in the middle, and 'Terminal 1234' on the right with a yellow border and background.
- User Authentication: Secure user login and registration.
- Account balance display: Persistent and real time user's balance display.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) to represent security and trust.
- Background color: Dark gray (#303030) to complement the navy blue in the primary header.
- Accent color: Golden yellow (#FFC107) for the 'Terminal' button and other important calls to action, providing contrast.
- Font: 'Inter' sans-serif for both headers to maintain clarity and readability; one size larger in the main 'LotoHub' title.
- Consistent alignment and spacing to ensure the elements are evenly distributed in both headers.
- Simple, clear icons for the home and menu buttons.
